## 기능

※ 로그인 필수

- .epub 파일 사이트 [z-library](https://ko.1lib.sk/)

- .epub 파일 업로드 기능
  - 업로드한 파일정보 DB에 저장
  - 파일명 혹은 도서명 기반으로 [도서관정보나루_API_Manual.pdf] API 연동하여 메타 데이터 조회
  - 메타 데이터 업데이트

- 업로드한 .epub 파일 목록 조회
- 업로드한 .epub 파일 삭제
- 업로드한 .epub 파일 읽기 (책처럼 보이는 UI)
  - 브라우저에서 .epub 파일 뷰어 기능
  - 브라우저 최적화 방법 고려

- 읽고 있는 책 체크기능
- 읽은 책 체크기능

- 특정책에 대한 감상평 작성
- 감상평 목록 조회

- 인상 깊은 구절 기록 (몇 페이지, 구절 내용)
  - 내가 현재 읽고 있는책의 해당 페이지를 입력하면 .epub 파일 데이터를 읽어서 그 페이지를 띄어주고
  - 해당 페이지에서 드래그에서 편하게 기록할 수 있도록
- 인상 깊은 구절 목록 조회

- 읽고 있는 책에서 따로 특정 몇일동안 완료처리 안되면 푸시알림
  - 아직 읽고있다고 체크하던지 혹은 다시 안읽고 있는책으로 변경
- 인상 깊은 구절 푸시알림 (주기 ??)

- 책 추천 !! [도서관정보나루_API_Manual.pdf]
  - 인기대출도서 조회
  - 마니아를 위한 추천도서 조회
  - 다독자를 위한 추천도서 조회

## 설정

- 몇일동안 완료안됐을떄 푸시알림 보낼지 '몇일' 설정 관리
- 인상 깊은 구절 푸시알림 '주기' 설정 관리
