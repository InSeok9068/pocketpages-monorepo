## 회원

- 회원가입
- 로그인/로그아웃
  - 일반 유저 로그인
  - 관리자 유저 로그인

## 자산 구분 및 리밸런싱 비율

- 현금 (10%)
  - 현금
  - 예적금
- 주식 (70%)
  - 성장 ETF (70%)
  - 배당 ETF (30%)
- 채권 (20%)
  - 국내채권
  - 해외채권

※ 연금은 담은 종목에 따라 정해짐

## 자산 데이터

- 토스 앱 자산 데이터 캡쳐 (스크롤 캡쳐)
- 자산 목록 (캡처 1)
- 주식 목록 (캡처 2)

## 기능

- 이미지 업로드하여 자산 데이터 입력
- 자산 목록 리스트로 표기
  - 자산 데이터 상세보기에서 해당 자산 데이터 수정 가능
- 원형 그래프로 자산 비율 표시
- 표 데이터로 자산 구분에 따른 현재 자산 비율과 목표비율과의 차이 표시
